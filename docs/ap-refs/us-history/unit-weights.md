## Generation strategy

When generating an MCQ for AP US History:
1. Pick a unit (weighted by exam weight — Unit 3-8 get more).
2. Pick ONE course theme from the 8 above.
3. Pick ONE reasoning process from {comparison, causation, continuity-change}.
4. Construct stimulus + 4 answer choices + correct answer + 1-line explanation tying back to the theme and process.

This tagging is how College Board builds the actual exam. Mirror it for authenticity.
