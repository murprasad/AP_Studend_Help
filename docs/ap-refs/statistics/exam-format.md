## Section I — Multiple Choice (PrepLion Practice scope)
- **40 questions · 90 minutes · 50% of total score**
- Delivered in Bluebook app
- Coverage: all 9 units, all 4 course skills
- **Format requirement for question generation:**
  - **4 answer choices (A-D)**, not 5
  - Calculator permitted throughout
  - Questions test: selecting method, analyzing data, using probability/simulation, statistical argumentation

## Section II — Free Response (out of scope for MCQ generation)
- **6 questions · 90 minutes · 50% of total score**
- 5 standard FRQs + 1 Investigative Task
- Handwritten on paper

## Four Course Skills (weights on exam)

| Skill | Description | Exam Weight |
|-------|-------------|-------------|
| 1. Selecting Statistical Methods | Select methods for collecting/analyzing data for inference | 15–23% |
| 2. Data Analysis | Describe patterns, trends, associations, relationships in data | 15–23% |
| 3. Using Probability & Simulation | Explore random phenomena | **30–40%** |
| 4. Statistical Argumentation | Develop explanation/justify conclusion using evidence | 25–35% |

**Probability & simulation dominates the MCQ section.** Generation should over-index on Skill 3 questions (binomial, normal, z-tests, confidence intervals, simulation).

## Three Big Ideas
1. **Variation and distribution**
2. **Patterns and uncertainty**
3. **Data-based predictions, decisions, conclusions**

## Quality Rubric for Generated MCQs

1. **Data-grounded**: every Q references specific numerical data, a scenario, or a distribution. No pure-definition recall questions.
2. **Skill-tagged**: map to one of the 4 skills above.
3. **Calculator-realistic**: if a Q requires computation, the numbers should be calculator-friendly, not arithmetic-heavy.
4. **4 options A-D** (not 5).
5. **Distractor quality**: common misconceptions (reversing H0/H1, confusing correlation with causation, misreading confidence-level vs significance-level, using σ when s is appropriate).
6. **Real-world context**: prefer applied scenarios (baseball stats, election polls, medical studies) over abstract number-crunching.
